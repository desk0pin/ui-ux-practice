What Is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It defines the structure of content using elements like headings, paragraphs, lists, and links.

An HTML document usually starts with <!DOCTYPE html>, then <html>, a <head> section, and a <body> section that contains visible content.

Common semantic tags you will see:

  • <header> for the top section of a page.
  • <nav> for navigation menus.
  • <main> for main content.
  • <section> for logical sections.
  • <footer> for page footer information.

Basic HTML Features

Below are some key features and topics that beginners should learn when starting with HTML.

Text & Headings

Use <h1> to <h6> for headings and <p> for paragraphs.

h1 h2 p span

Links & Images

Use <a> for links and <img> for images (with a valid src URL).

a href img alt

Lists & Tables

Use ordered lists (<ol>), unordered lists (<ul>), and tables (<table>).

ul ol li table

Forms

Use <form> with inputs, labels, and buttons to collect user data.

form input textarea button

Minimal HTML Structure Example

Here is a small example of the core HTML structure taken from a typical beginner tutorial. You can compare it with the full page code you are reading now. 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My First Page</title>
</head>
<body>
    <h1>Hello World!</h1>
    <p>This is my first web page.</p>
</body>
</html>

Practice: Create a new HTML file with only this minimal structure, then gradually add sections like on this full example page to train your understanding step by step.
